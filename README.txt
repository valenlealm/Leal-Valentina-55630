# Proyecto Gestión de Ventas
Este proyecto es una aplicación de gestión de datos desarrollada en Python utilizando el framework Django. Tiene tres modelos: cliente, producto y vendedor. También incluye formularios para insertar datos en los modelos y para buscar información en la base de datos.

## Modelos
Cliente
     nombre
     email 
     telefono 

Vendedor
     nombre
     email 
     telefono 

Producto
     nombre
     precio

## Funcionalidad
En la página principal, encontrarás enlaces a las secciones de productos, clientes y vendedores.
Podrás consultar clientes, productos y vendedores existentes
Podrás agregar nuevos clientes y consultar clientes en un formulario de búsqueda que te generarà resultados que contengan caracteres incluidos en la búsqueda.
